# Movie App

This is an Movie website

## Getting Started

This app tested on node.js v20.17.0

First, install the dependencies:

```bash
npm install
```

Then, run dev server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
